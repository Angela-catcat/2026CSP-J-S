# [CSP-J 2025] 拼数

## 题意概括

给定一个字符串 $s$，其中只包含小写英文字母和数字，并且保证至少包含一个 $1 \sim 9$ 的数字。

我们可以从 $s$ 中选择任意多个数字字符，每个位置上的数字字符最多使用一次，然后把选出的数字字符任意重排，拼成一个正整数。

要求输出可以拼出的最大正整数。

注意：如果字符串中有多个相同数字字符，它们可以分别使用。例如 `1a01b` 中有两个字符 '1'，所以两个 `1` 都可以用，但不能凭空多用第三个 `1`。

## 解题策略

两个正整数比较大小的时候，通常先看位数，所以为了让答案最大，每个数字字符都应该使用。因为题目保证至少有 $1 \sim 9$ 中的一个字符，因此一定可以把某个非零位放在最高位，无前导零的情况发生。

位数相同的情况下，会从最高位开始逐个比较，数字越大，整个数越大，所以将所有数字字符从大到小进行排序，然后输出。


### 代码一

使用桶计数统计每个数字出现次数，再从大到小输出。

```cpp
#include <bits/stdc++.h>
using namespace std;

int cnt[10];

int main() {
    string str;
    cin >> str;

    for (auto x : str) {
        ++cnt[x - '0'];
    }

    for (int i = 9; i >= 0; i--) {
        while (cnt[i]) {
            cout << i;
            --cnt[i];
        }
    }

}
```

### 代码二

将字符串排序后，所有数字字符会按从小到大的顺序排列。

我们提取出这些数字字符，得到升序排列的数字串。由于最大整数需要高位尽可能大，所以将该数字串翻转，得到降序排列后的结果，即为答案字符串。
 
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string x;
    cin >> x;

    sort(x.begin(), x.end());

    string ans = "";
    for (auto it : x) {
        if (it >= '0' && it <= '9') ans += it;
    }

    reverse(ans.begin(), ans.end());

    cout << ans << "\n";
}
```

### 扩展｜前导 0 的处理

`find_first_not_of` 是 C++ 中 std::string 类的一个成员函数，它的核心作用是：在字符串中从前往后查找，返回第一个「不属于」指定字符集合的字符的下标。

如果字符串中的所有字符都属于指定的字符集合（或者字符串为空），则返回 string::npos（通常为 -1）。

借助该函数，我们可以快速找到第一个非零数字字符的下标，再结合 `substr` 函数从该位置进行切割，即可得到丢掉前导 0 后的字符串。

```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string s = "000123";
    
    // 查找第一个不是 '0' 的字符的位置
    int pos = s.find_first_not_of('0');
    
    if (pos == -1) {
        cout << "字符串全是 '0'";
    } else {
        cout << "第一个非 '0' 字符在位置: " << pos << endl;
        cout << "去除前导零后的字符串: " << s.substr(pos);
    }
    return 0;
}
```

也可以使用三目运算符进行压行：
```cpp
cout << (pos == -1 ? "0" : s.substr(pos)) << '\n';
```

该写法可以处理：
- 全是 0 的情况，例如 `0` 和 `000`
- 有前导 0 的情况 ，例如 `000123`
- 有尾随 0 的情况，例如 `10`
